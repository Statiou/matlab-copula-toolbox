# MATLAB Copula Toolbox — All-in-One

**Statement of need.** Dependence modelling with copulas + clustering in one MATLAB toolbox, with both GUI and scriptable API.

## Features
- Gaussian/t/Archimedean (Clayton/Frank/Gumbel) fits; U-mapping & diagnostics
- Clustering on X or U (k-means, GMM, hierarchical, DBSCAN) + per-cluster copulas
- Mixture-of-copulas (2D), GAS τ forecasting, CoVaR/CoES, bootstrap, CV
- Single-file GUI: `fitcopulatoolbox_allinone.m`

## Install
MATLAB R2023b+ (Statistics & Machine Learning Toolbox).  
Clone → add folder to path.
